### Task01 - [赛题理解](/nbs/Task01-赛题理解.ipynb) 🍻

- 赛题数据：新闻文本 ( 按照**字符**级别匿名处理 ) / 14个类别
- 数据构成：训练集 (20w) / 测试集A (5w) / 测试集B (5w)
- 评价指标：f1_score的均值
- 解题思路
  - 思路1：TF-IDF + 机器学习分类器
  - 思路2：FastText
  - 思路3：word2vec + 深度学习分类器
  - 思路4：Bert
